# 5-GlobalData 全局数据
::: danger
+ 良药与毒药的区别在于剂量
+ 封装在函数内可以保证可控性
:::

## 定义
在全局都可以被获取和更改的变量

## 表现形式
+ 全局变量
+ 类变量
+ 单例 (singleton)

## 方案
+ 封装在函数内
